# deepstack-web-ui
Unofficial web UI for working with Deepstack.
Run the app:
```
$ python3 app/server.py serve
```

## Development
Use venv:
```
$ python3 -m venv venv
$ source venv/bin/activate
$ pip install --upgrade -r requirements.txt
```
